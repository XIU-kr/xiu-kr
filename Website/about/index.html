XIU

호기심 많은 풀스택 개발자

필요한 것이 없으면 직접 만듭니다. 웹, 모바일, AI, 게임 서버, 인프라까지 — 도메인의 경계 없이 문제를 해결하는 개발자 XIU입니다.

소개

저는 특정 분야에 머무르지 않는 개발자입니다. 처음 HTML로 웹페이지를 만들던 날부터 지금까지, "이건 내가 직접 만들 수 있지 않을까?"라는 질문이 저를 움직이게 합니다. 그 질문 하나가 AI 이미지 편집기를 만들게 했고, 게임 서버 플러그인을 개발하게 했으며, 교회를 위한 기술 솔루션을 설계하게 만들었습니다.

Vora는 그 철학의 결과물 중 하나입니다. Mac, Windows, Linux 모두에서 돌아가는 크로스플랫폼 AI 이미지 에디터로, Tauri 위에 React 프론트엔드와 Python AI 파이프라인을 올려 포토샵 수준의 레이어 시스템과 Big-LaMa 인페인팅을 구현했습니다. 설치 후에는 완전 오프라인으로 동작하며, 작업물이 외부 서버로 전송되지 않습니다.

게임 커뮤니티에서는 CS2 서버를 직접 운영하며 2500명 이상의 멤버가 활동하는 커뮤니티를 키워왔습니다. 서버 관리에 필요한 플러그인이 없으면 C#으로 직접 개발합니다. 매치 시스템, 통계 추적, 관리 도구 등 커뮤니티가 필요로 하는 것을 코드로 만들어 제공합니다.

동탄명성교회를 위한 기술 솔루션도 오랫동안 개발해 왔습니다. 예배 PPT를 자동으로 생성하는 Phos, 교인 관리 시스템 Poima, 그리고 Jetpack Compose로 만든 모바일 앱까지. 기술이 사람들의 일상에 실질적인 도움이 되는 순간이 가장 보람 있습니다. 서버 인프라 역시 직접 설계하고 운영합니다. Cloudflare와 OPNsense를 활용한 다계층 보안 아키텍처로 모든 서비스를 안정적으로 제공하고 있습니다.

개발 철학

🔧

문제를 발견하면 직접 만듭니다

기존 도구가 만족스럽지 않으면 직접 만듭니다. 필요에 의해 시작된 프로젝트가 결국 다른 사람들에게도 가치를 만들어냅니다. Vora도, Phos도, 게임 플러그인도 모두 "이게 없네, 그러면 내가 만들자"에서 시작되었습니다.

🎯

끝까지 만듭니다

기획부터 디자인, 개발, 배포, 운영까지 전 과정을 직접 수행합니다. 프로토타입에서 멈추지 않고 실제 사용자가 쓸 수 있는 프로덕션 수준까지 완성합니다. 반쯤 만들어진 프로젝트는 의미가 없다고 생각합니다.

🖥

모든 것을 직접 운영합니다

코드만 작성하고 끝나지 않습니다. 서버 구축, 네트워크 설계, 보안 설정, 모니터링까지 인프라 전반을 직접 관리합니다. 코드가 실행되는 환경을 이해해야 더 좋은 소프트웨어를 만들 수 있습니다.

🌐

오픈소스로 공유합니다

대부분의 프로젝트를 GitHub에 공개합니다. 제가 문제를 해결한 방법이 다른 개발자에게도 도움이 되길 바랍니다. 혼자 쌓아두는 지식보다 함께 나누는 코드가 더 큰 가치를 만든다고 믿습니다.

개발 여정

웹 개발의 시작

HTML, CSS, JavaScript로 첫 웹사이트를 만들며 개발의 세계에 발을 들였습니다. PHP와 MySQL을 독학하며 동적 웹 애플리케이션의 구조를 이해하기 시작했고, 화면에 무언가를 직접 만들어낸다는 경험에 완전히 빠져들었습니다.

교회 기술 솔루션

동탄명성교회의 실무 필요를 해결하기 위해 개발을 시작했습니다. 찬양 가사를 자동으로 PPT로 변환하는 Phos를 만들었고, 교인 정보를 체계적으로 관리하는 Poima 시스템을 구축했습니다. 기술이 실제 사람들의 수고를 덜어주는 경험이었습니다.

게임 커뮤니티 구축

CS2 커뮤니티 서버를 런칭하고 2500명 이상의 활발한 커뮤니티로 성장시켰습니다. C#으로 커스텀 플러그인을 개발하며 매치메이킹, 플레이어 통계, 서버 관리 도구를 직접 만들었습니다. 사용자의 피드백을 실시간으로 받으며 빠르게 개선하는 법을 배웠습니다.

모바일 개발

Jetpack Compose를 활용해 Android 앱 개발에 도전했습니다. 교회용 모바일 앱과 유틸리티 앱 Quon을 개발하며, 모바일 환경에서의 사용자 경험 설계와 네이티브 성능 최적화를 깊이 있게 경험했습니다.

AI와 크리에이티브 도구

Vora를 통해 AI 기반 이미지 편집의 가능성을 탐구했습니다. Tauri로 Mac, Windows, Linux를 모두 지원하는 단일 바이너리를 만들고, Big-LaMa 모델을 로컬에서 실행하도록 통합해 설치 후 완전 오프라인으로 동작하는 포토샵급 데스크톱 에디터를 완성했습니다. AI를 단순히 쓰는 것이 아니라 제품의 사용 경험에 녹여내는 과정이었습니다.

인프라 설계

Cloudflare와 OPNsense를 결합한 다계층 보안 서버 아키텍처를 직접 설계하고 운영하기 시작했습니다. 모든 서비스의 안정적인 운영을 위해 네트워크 보안, SSL 관리, 트래픽 최적화까지 인프라 전반을 책임지고 있습니다.
