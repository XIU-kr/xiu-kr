Tailwind CSS

🛠

AI / ML

🧠

Big-LaMa

대규모 이미지 인페인팅 모델. Vora의 핵심 기능인 배경 제거와 이미지 복원에 사용합니다.

📷

ML Kit

Google의 온디바이스 ML 프레임워크. Quon Android의 실시간 QR 코드 스캔 기능에 활용합니다.

📹

CameraX

Android 카메라 프레임워크. 실시간 이미지 처리와 카메라 기반 기능 구현에 사용합니다.

브라우저와 모바일에서 AI를 직접 실행하는 것에 관심이 많습니다. 서버에 의존하지 않는 온디바이스 AI와 WebAssembly 기반 추론으로, 사용자에게 더 빠르고 프라이버시를 보장하는 경험을 제공하는 방향을 추구합니다.

인프라 & DevOps

🐳 Docker

모든 서비스를 컨테이너로 격리하여 운영. 개발 환경과 프로덕션 환경의 일관성을 유지합니다.

☁ Cloudflare

DNS, CDN, DDoS 방어, SSL 인증서 관리. 모든 도메인의 트래픽을 Cloudflare를 통해 보호합니다.

🛡 OPNsense

오픈소스 방화벽. 홈 서버 네트워크의 보안과 트래픽 관리를 담당합니다.

🎮 Pterodactyl

게임 서버 관리 패널. CS2 등 게임 서버를 웹 UI로 편리하게 관리합니다.

🔄 PM2

Node.js 프로세스 매니저. 무중단 운영, 로그 관리, 자동 재시작을 처리합니다.

🌐 Nginx

리버스 프록시와 웹 서버. 포트폴리오 사이트를 포함한 모든 웹 서비스의 진입점입니다.
